This Website

Project Background

This project was an assignment for my Technology Communications (ME 1100) course at UConn Fall 2025. It was our final project of the semester, meant to accumulate everything we had learned, and turn it into something we can utilize professionally.

Objective(s)

The main objective of this project was to create a professional portfolio that laid out important information about us as students/professionals. This included an "About Me" section, access to our resume, as well as a collection of experience and/or projects that we have done. My own personal objectives for this project were to implement my knowledge in HTML and CSS as another way to present my skills.

Process

The two biggest portions of this project were building the website HTML and CSS from scratch, and organizing my collection of projects. The website building was the easiest part: I start by building a "master" HTML document that includes all of the code for universal portions of the website, such as the header and background image, that will be displayed on every page. Once that is in order I can simply duplicate the document for each page of the website that I wish to create, then simply adding new code to format each unique page. The most difficult part of the process was organizing my project collection. Many of them are personal projects, and I don't usually neatly document them, or document them at all often times. Because of this, I had to spend an exhausting amount of time finding images and documents relating to my completed projects, and then bringing everything together so that I can spend more time writing about said projects.

Conclusion

Overall, choosing to build this website from scratch as part of the assignment, although very time consuming, was a worthwhile decision. It felt nice to delve back into web development and other skills that I don't tend to make frequent use of.
